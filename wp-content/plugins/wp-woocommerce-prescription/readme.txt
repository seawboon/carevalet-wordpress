WordPress WooCommerce Prescription facilitates the customer to buy medicine by uploading prescription for selected medecine and admin will only allow those customer to buy medicine who will upload doctor's prescription.

== Description ==

The admin can allow customers to buy medicine when they will upload prescription for that medicine. Admin can choose which products can buy after uploading prescription which products don't need any prescription accordinglly.

== Features ==
* The store owner can allow customers to buy medicine only when they have doctor's prescription.
* The store owner can choose which products can buy after uploading prescription which products don't need any prescription.
* The user can easily upload their doctor's prescription and buy medicine online.


== Installation ==

1. Install the Woocommerce Prescription
2. Activate the plugin
3. Go to the Woocommerce Prescription
